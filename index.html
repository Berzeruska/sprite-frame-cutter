Como usar

1. Clique em "Carregar Sprite" ou arraste a imagem aqui
2. Clique e arraste no canvas para marcar um frame
3. Dê um nome ao frame (ex: idle_0, walk_0…)
4. Repita para cada animação
5. Clique em "Exportar Frames" para baixar os PNGs
